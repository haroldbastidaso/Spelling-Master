# 🐝 Spelling Bee Academy - Duo Edition

Una aplicación web interactiva diseñada para el aprendizaje del abecedario y el deletreo en inglés, inspirada en la metodología de gamificación y con la estética de Duolingo. Proyecto desarrollado para la comunidad educativa del **Instituto Champagnat**.

## 🚀 Características

* **6 Niveles de Dificultad:** Desde reconocimiento visual del alfabeto hasta escritura de deletreo fonético complejo.
* **Motor de Voz Real:** Integración con `Web Speech API` optimizada para voces en inglés americano (`en-US`).
* **Gamificación:** Sistema de feedback visual con el personaje de Duo, sonidos de acierto/error y temporizadores de presión.
* **Diseño Responsive:** Adaptado para funcionar en computadoras, tablets y dispositivos móviles.
* **Cálculo de Notas:** Generación de calificación automática sobre una escala de **5.0**.

## 🎮 Niveles Incluidos

1.  **Alfabeto Visual:** Asociación de letra con su fonética.
2.  **Oído Alfabeto:** Escuchar la letra y seleccionar el carácter correcto.
3.  **Oído Deletreo:** Escuchar la letra y seleccionar la fonética correcta.
4.  **Palabras con 'A':** Práctica de deletreo con vocabulario inicial.
5.  **Palabras con 'B':** Práctica de deletreo intermedio.
6.  **Maestro Escritor:** Reto máximo donde el usuario debe escribir el deletreo fonético completo.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS3:** Diseño personalizado, variables CSS y animaciones.
* **JavaScript (Vanilla):** Lógica del juego, manejo del DOM y control de estados.
* **Web Speech API:** Para la síntesis de voz en tiempo real.

## ⚙️ Instalación y Uso

No requiere servidor ni dependencias externas. 

1.  Clona el repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/spelling-bee-duo.git](https://github.com/tu-usuario/spelling-bee-duo.git)
    ```
2.  Abre el archivo `index.html` en tu navegador (preferiblemente **Google Chrome** para una mejor experiencia de voz).

## 🎙️ Nota sobre la Síntesis de Voz
Para una experiencia óptima, asegúrate de tener instalado el paquete de idioma **English (United States)** en la configuración de tu sistema operativo. El programa buscará automáticamente voces de alta calidad como "Google US English".
